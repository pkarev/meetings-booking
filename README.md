# meetings-booking

Empty project.

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To run dev server with live-reload

```sh
npm start
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

## Running

```sh
node dist/bundle.js
```
